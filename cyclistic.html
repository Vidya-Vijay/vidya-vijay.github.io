Pedaling Toward Insights: Cyclistic Bike-Share Analysis

Google Analytics Capstone Project

Project Summary

As part of the Google Data Analytics Capstone Project, I analyzed Cyclistic’s Q1 2020 bike-share data to uncover differences between casual riders and members. Using SQL Server for data cleaning and analysis, and Power BI for visualization, I identified key trends in trip volume, ride duration, and time-of-day patterns. My findings informed targeted marketing strategies and operational improvements, with a focus on converting casual riders into loyal members.

Tools & Techniques

  • Data Cleaning & Analysis: SQL Server Management Studio (SSMS)
  • Visualization: Power BI
  • Skills Applied: Data wrangling, aggregation, time-based analysis, DAX, data storytelling

Key Tasks

  • Imported and cleaned raw CSV datasets in SQL Server.
  • Wrote SQL queries for summary statistics, time trends, and station analysis.
  • Designed interactive Power BI dashboard using DAX measures and custom visuals.
  • Developed insights and recommendations from the data to support business decisions.

Key Insights

  • Members account for 88.64% of rides (378K trips), and casual riders only 11.36% (48K trips).
  • Member usage dominates each month, with January leading at 136K rides.
  • Casual ridership is relatively low but shows growth from January (8K) to March (28K), hinting at seasonal patterns or warming weather.
  • Casual riders take longer trips (20.3 min average) vs. members (10.5 min), suggesting leisure usage.
  • Members peak midweek (commuting patterns), casual riders peak on weekends (recreational usage).
  • Start time analysis reveals two strong peaks for members (8 AM & 5 PM) and flat usage for casuals, with slight afternoon activity.

    • Recommendations

    1. Target casual riders with weekend leisure promotions and convert them into members.

    2. Increase bike availability in key stations during weekday peaks and weekend leisure times.

    3. Focus marketing efforts on stations with high casual riders activity.
